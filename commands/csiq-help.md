# /csiq-help

Display all available sfdx-iq slash commands and CLI tools with descriptions.

## Workflow

1. Use Glob to find `**/sfdx-iq/scripts/csiq.js`
2. Run via Bash: `node <found-path> help`
3. Display the output to the user
