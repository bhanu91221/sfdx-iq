# /status

Show claude-sfdx-iq plugin status, installed component counts, and connected org info.

## Workflow

1. Use Glob to find `**/claude-sfdx-iq/scripts/status.js`
2. Run via Bash: `node <found-path>`
3. Display the output to the user
